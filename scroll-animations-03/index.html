Animation Ranges

The animation-range property determines which part of the view timeline triggers the animation.

Fine-tuning with keywords

Use keywords like entry, exit, cover, and contain to control when an animation starts and ends.

animation-range: cover

Starts as the top enters, finishes as the bottom leaves.

animation-range: contain

Starts when fully inside, ends when it starts to leave.

animation-range: entry

Only animates while entering the viewport.

animation-range: exit

Only animates while leaving the viewport.
